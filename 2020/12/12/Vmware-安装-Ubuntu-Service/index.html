LINUX下添加512M交换分区

参考链接，交换分区的大小应该怎么设置
原文链接

1、root用户登录

2、创建存储文件：

dd if=/dev/zero of=/swapfile1 bs=1024 count=524288/1048576 (根据需要选512M或1024M)

3、交换文件的安全处理

chown root:root /swapfile1
chmod 0600 /swapfile1

4、设置一个Linux的交换区域

mkswap /swapfile1

5、启用交换文件

swapon /swapfile1

6、修改/etc/fstab文件

vim /etc/fstab

在最后追加一行：

/swapfile1 none swap sw 0 0

7、重启reboot验证交换分区的大小

free -h

LINUX下禁止ping命令的使用

在黑客入侵寻找对象时，大多都使用Ping命令来检测主机，如果Ping不通，水平差的“黑客”大多就会知难而退。
事实上，完全可以造成一种假相，即使我们在线，但对方Ping时也不能相通，这样就能躲避很多攻击。

以root进入Linux系统，然后编辑文件icmp_echo_ignore_all

vi /proc/sys/net/ipv4/icmp_echo_ignore_all

将其值改为1后为禁止PING
将其值改为0后为解除禁止PING

直接修改会提示错误:

WARNING: The file has been changed since reading it!!!
Do you really want to write to it (y/n)?y
“icmp_echo_ignore_all” E667: Fsync failed
Hit ENTER or type command to continue

这是因为 proc/sys/net/ipv4/icmp_echo_ignore_all
这个不是真实的文件
如果想修改他的数值可以echo 0 或 1到这个文件

（即echo 1 > /proc/sys/net/ipv4/icmp_echo_ignore_all ）。要是想永久更改可以加一行

net.ipv4.icmp_echo_ignore_all=1

到配置文件/etc/sysctl.conf里面

克隆虚拟机配置ip

获取虚拟机网络NAT配置信息

虚拟机网络选择nat模式，从虚拟机的nat网络配置里面获取到如下信息:

  1. 子网ip：192.168.74.0
  2. 子网掩码：255.255.255.0
  3. 默认网关：192.168.74.2 (在nat设置里面获取)
  4. DHCP-起始：192.168.74.128
  5. DHCP-结束：192.168.74.254
